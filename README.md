## What is LiteLLM?

LiteLLM is a **unified interface** that provides:

1. **Python SDK** - Direct library integration in your code
2. **Proxy Server (AI Gateway)** - Central service for multi-LLM access
3. **100+ LLM Providers** - OpenAI, Anthropic, Azure, Bedrock, Ollama, and more
4. **Key Features** - Cost tracking, guardrails, load balancing, virtual keys, caching
